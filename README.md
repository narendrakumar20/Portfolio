# 🚀 3D Developer Portfolio Website

A modern, interactive, and high-performance **3D Developer Portfolio** built with **React**, **TypeScript**, **Three.js**, **GSAP**, and **WebGL**.

Designed to showcase projects, skills, and experience through immersive 3D visuals, smooth animations, and a fully responsive user experience.

🌐 **Live Demo:** https://narendrakumar20.github.io/Portfolio/

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/narendrakumar20/Portfolio.git
cd Portfolio
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
